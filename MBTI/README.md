# 초등학생용 MBTI 성향 검사 앱

HTML, Tailwind CSS CDN, JavaScript만 사용한 정적 웹앱입니다.

## 실행 방법
1. `index.html` 파일을 브라우저로 엽니다.
2. 이름과 학년을 입력합니다.
3. 질문에 모두 답합니다.
4. `저장하고 분석하기` 버튼을 누르면 결과 페이지가 나옵니다.

## 구성 파일
- `index.html`: 화면 구조와 Tailwind 디자인
- `script.js`: 질문 생성, 응답 저장, MBTI 계산, 결과 출력

## 참고
이 앱은 재미와 자기이해를 위한 간단한 성향 검사입니다.
전문 심리검사가 아니므로 결과를 절대적인 성격 판단으로 사용하지 않는 것이 좋습니다.
